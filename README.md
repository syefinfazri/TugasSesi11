# Penjelasan

# Aktivitas 1 - Selectiion Sort Ascending
- a. Data sebelum sorting: 20, 18, 25, 34, 6
- b. Data sesudah sorting: 6, 18, 20, 25, 34
- c. Apakah minIndex berpindah di setiap iterasi? Tidak selalu. minIndex hanya berubah jika ditemukan nilai yang lebih kecil dari elemen sebelumnya.
# Aktitas 2 - Selection Sort Descending
- a. Data sebelum sorting: 20, 18, 25, 34, 6
- b. Data setelah sorting: 34, 25, 20, 18, 6
- c. Perbedaan logika ascending vs descending: Ascending mencari nilai terkecil, sedangkan descending mencari nilai terbesar di setiap iterasi.
